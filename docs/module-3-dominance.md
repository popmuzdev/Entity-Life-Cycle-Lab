# Модуль 3: Dominance (Доминирование)

**Invisible Leader Protocol**

Как стать единственным очевидным выбором в своей нише, даже без Википедии и больших бюджетов.

## Что такое Invisible Leader

Когда ИИ-ассистент пользователя отвечает на запрос в вашей нише — он называет **вас** первым.  
Без агрессивной рекламы. Без миллионов просмотров. Только за счёт силы сущности.

## Как достичь доминирования (ключевые практики)

### 1. Локальное/нишевое семантическое ядро
- Определите свою узкую категорию (например: «маркетолог для артистов», «Entity Engineering для личных брендов» и т.д.).
- Создайте мощные триплеты именно для этой ниши.

### 2. Постоянное усиление Authority
- Регулярные публикации на vc.ru с чёткими смыслами.
- Кросс-ссылки между вашими материалами.
- GitHub + DOI как «твёрдые» сигналы.

### 3. Контент, который ИИ любит цитировать
- Практические чек-листы
- Реальные кейсы и факапы
- Глубокие размышления + конкретные инструменты

### 4. Мониторинг позиций
Регулярно проверяйте:
- Что отвечает Gemini / ChatGPT на запросы вашей ниши?
- Вы в топ-3 источников или нет?
